# Visão Geral do Git Scope

<Note>
  Git Scope é uma ferramenta de análise e filtragem de diffs do Git, projetada para
  melhorar a produtividade e qualidade do processo de revisão de código.
</Note>

## Objetivos

<CardGroup cols={2}>
  <Card title="Produtividade" icon="rocket">
    Reduzir o tempo gasto em revisões de código focando nas mudanças relevantes
  </Card>
  <Card title="Qualidade" icon="shield-check">
    Melhorar a qualidade das revisões com análise contextual e filtros inteligentes
  </Card>
  <Card title="Automação" icon="robot">
    Automatizar tarefas repetitivas no processo de revisão de código
  </Card>
  <Card title="Integração" icon="puzzle-piece">
    Integrar com ferramentas existentes no fluxo de desenvolvimento
  </Card>
</CardGroup>
