# Git Scope

<Note>
  Git Scope é uma ferramenta moderna e eficiente para análise e filtragem de diffs do Git,
  permitindo uma melhor compreensão e controle sobre as mudanças em seu código.
</Note>

## Começando

<Steps>
  <Step title="Instalação">
    ```bash
    pip install git-scope
    ```
  </Step>
  <Step title="Configuração">
    ```bash
    git-scope init
    ```
  </Step>
  <Step title="Uso Básico">
    ```bash
    git-scope analyze HEAD~1
    ```
  </Step>
</Steps>

## Recursos Principais

### Análise de Diff

<CodeGroup>
  <CodeBlock title="Comando">
    ```bash
    git-scope analyze --filter "*.py" --exclude "tests/*" HEAD~1
    ```
  </CodeBlock>
</CodeGroup>
